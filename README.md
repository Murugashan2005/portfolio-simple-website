# Personal Portfolio Website

A simple and responsive personal portfolio website built using **HTML** and **CSS**. This project showcases a clean and beginner-friendly design with easy navigation.

## Features

* 🏠 Home page with personal introduction
* 📞 Contact page for communication details
* 📱 Responsive and clean user interface
* 🎨 Built using only HTML and CSS

## Technologies Used

* HTML5
* CSS3

## Project Structure

```
portfolio-website/
│── index.html
│── contact.html
│── style.css
└── assets/
```

## Purpose

This project was created to practice front-end web development fundamentals and demonstrate basic website design using HTML and CSS without any frameworks or JavaScript.
